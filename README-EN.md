# WebChecker
## Description
Automatically batch check if web pages online.

## How to use
./WebChecker.sh website.txt

## Web List Format
```
www.google.com
www.fackbook.com
www.youtube.com

```

## Example Result
```
-----Start testing-----

www.google.com OK
www.fackbook.com OK
www.youtube.com OK

-----Test end-----
```
